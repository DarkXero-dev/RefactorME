65% // Gasket // Hot-swap

One block of aluminium, tuned to sound like this

Nyx65 is a 65% mechanical keyboard machined from a single billet of 6063 aluminium. Gasket-mounted, hot-swappable, and voiced for a deep, muted thock straight out of the box.

// BUILT TO BE HEARD //

01

Unibody Aluminium

CNC-milled from one 1.4kg block of 6063 aluminium, then bead-blasted and anodised. No seams, no rattle, just a dense case that stays put on the desk.

02

Gasket Mount

The plate floats on silicone gaskets, so every keypress gives a little before it bottoms out. Softer typing feel, deeper acoustics, none of the hollow ping.

03

Hot-swap Sockets

Five-pin hot-swap sockets across the whole board. Pull a switch, drop in a new one, no soldering iron. Ships pre-lubed with tactile switches, ready the second it lands.
